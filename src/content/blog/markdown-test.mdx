# Markdown Examples

This post demonstrates various markdown and MDX features.

## Text Formatting

**Bold text** and *italic text* are supported.

## Lists

Unordered list:
- Item 1
- Item 2
  - Nested item
  - Another nested item
- Item 3

Ordered list:
1. First item
2. Second item
3. Third item

## Code Blocks

```javascript
function hello() {
  console.log("Hello, world!");
}
```

Inline `code` is also supported.

## Blockquotes

> This is a blockquote.
> It can span multiple lines.

## Custom Components

  This is a custom note component!
</Note>

</Note>
